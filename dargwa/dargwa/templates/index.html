{% trans 'даргинских языков' %}

{% trans 'ДарЛинг' %}{% trans ' – это лексикографическая база данных, содержащая лексику нескольких даргинских языков и диалектов. В базе возможен поиск лексем одного или нескольких языков по различным типам запросов – по даргинскому слову, по русскому или английскому переводу. Доступен также поиск синонимов, когнатов (слов, восходящих к одному источнику) и слов, содержащих определенные морфемы.' %}

{% trans 'Даргинские словоформы представлены в кириллической записи (на основе даргинской орфографии) и в транскрипции. Помимо значения слов, словарные статьи содержат грамматическую информацию (различные формы слов, модели управления глаголов, информация о грамматическом классе имен). Для всех слов приводится информация об источнике данных.' %}

{% trans 'Первая версия лексикографической базы данных была создана в 2018 году. В 2022 году созданная ранее база была реорганизована. В настоящее время в базе содержится более 2000 компаративных серий, в том числе более 300 глагольных корней. Для создания и пополнения базы использованы два типа источников данных: опубликованные словари и грамматики (в том числе онлайн-ресурсы) и полевые материалы, собранные авторами и их коллегами. Для включения в базу все данные представлены в едином формате, слова разных языков и диалектов организованы в компаративные серии.' %}

{% trans 'В настоящее время в базе доступны словари следующих идиомов: акушинского, тантынского, ицаринского, муиринского, кадарского, мегебского и кункинского. База постепенно пополняется новыми данными. В частности, готовится лексикон урахинского и кубачинского диалектов, а также данные из «Диалектологического словаря даргинского языка» С. М. Темирбулатовой (2022).' %}

{% trans 'Над базой работали:' %}
{% trans 'Н. Р. Сумбатова (разработка проекта базы и формата словарных статей; составление компаративных серий; сбор и представление данных)' %}
{% trans 'Элли Сокур (программирование текущей версии базы)' %}
{% trans 'В. А. Позднякова (разработка формата словарных статей для глаголов; программирование первой версии базы)' %}

{% trans 'Ицаринский: словник из работы (Sumbatova, Mutalov 2003) и словник, собранный Р. О. Муталовым (не опубликован); данные для базы подготовили И. В. Калякин и Н. Р. Сумбатова.' %}
